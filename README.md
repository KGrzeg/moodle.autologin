# Moodle Autologin

Automatically login into moodle platform of my college using puppeteer for future automations purpose.

# Install dependencies

```sh
$ npm i
```

# Run

```sh
$ npm start -- <argumens>
#or
$ ./index.js <arguments>
#or
$ node index <arguments>
```

# Usage

Try

```sh
$ ./index.js --help
```
